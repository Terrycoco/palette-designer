# Palette Designer

A color designer project using ReactJS, Webpack, Redux, Redux-Thunk

###Also Includes###
* Testing with Mocha & Chai
* react-router
* Example Files


###Getting Started###


```
	> git clone git@github.com:terrycoco/palette-designer.git <YourProjectName>
	> cd  <YourProjectName>
	> npm install
	> npm start
	> open a browser to localhost:8080
```
